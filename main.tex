\section{Introduction}

The Hurwitz zeta function is defined as
\[
\zeta(s,x) =\sum _{i=0}^{\infty }\frac  {1}{(x+i)^{s}}, \qquad s > 1, \ x > 0.
\]
We want to approximate it well.

Some series representations include the Newton series:
\[
